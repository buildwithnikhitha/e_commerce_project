A Python-based console application for managing products, users, orders, and billing using JSON file storage.

---

## ✨ Features

- 📦 Add Products
- 👀 View Products
- ✏️ Update Products
- ❌ Delete Products
- 👤 User Registration
- 🔐 User Login
- 🛒 Place Orders
- 🧾 Generate Bills
- 📜 View Order History
- 💾 Store Data using JSON Files

---

## 🛠️ Technologies Used

- Python 3.x
- JSON
- VS Code
- Git & GitHub

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

### Installation

bash
git clone https://github.com/buildwithnikhitha/e_commerce_project.git
cd e_commerce_project
python main.py
